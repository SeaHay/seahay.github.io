Projects I've made/worked on:

Here's a list of projects that I've either made, or worked on. They include both GitHub and non-GitHub projects, and will also include things that may not yet be done/got on a long-term hiatus



Hangman

A python script I'm writing. It does a two player hangman game supporting phrases that include non-alphanumerical characters! (They just won't be able to be solved for...) The project started when we did this as an assignment, but instead it was only one iteration each time, you could guess multiple letters, there were no lives, and to top the cake off, you'd have to type in the answer as well to start it. Now a check system is in place, there are classic and dynamic lives, and at some point, I hope to have a word bank to draw phrases from for a single player mode.



Idea Consultancy

Another python script, this one built for giving answers for any questions you might have when working on any sort of project. (Warning! stupid creation ahead!)



Angle Computation

A script that takes an angle and outputs coterminal, quadrant, reference and number of rotations.
Available for use in a console output text menu ('textMenu()'), as well as a Tkinter GUI ('graphicalMenu()')
The program only supports negative and positive whole number degree angles, but future support may come to angles with decimals, as well as support for radians.
